Introduction to Keyboard Navigation and Accessibility

Background

These are the materials that my colleague Tamsyn Smith and I put together for running an interactive webinar on keyboard navigation while also introducing the basic principles of accessibility. The aim is to run this session within an in-person workshop or in interactive webinar format.

As is often the case I include many hidden slides in the presentation which include much more in-depth detail and developer content.
